# v1.1 第二界面优化验收方案

## Context

对「鸭子所在地太阳视运动轨迹」第二界面进行四项优化：

1. 中心鸭子替换为第一界面同款 3D 鸭子模型
2. 去除天球多余网格线，地面改为高精度地图纹理
3. 太阳低于地平面时隐藏（不透出地面以下）
4. 太阳位置变化时鸭子身上投射动态阴影

---

## 验收条件

### AC-1: 中心鸭子建模

- [ ] 第二界面中心显示与第一界面同款黄色 3D 鸭子模型
- [ ] 鸭子朝向：头 = +Z（北）、尾 = -Z（南）、左 = -X（西）、右 = +X（东）
- [ ] 鸭子大小适中（可参考第一界面 scale，约放大 1.5~2x）
- [ ] 移除原来的黄色小球 + 十字标

### AC-2: 地面与穹顶视觉

- [ ] 地面圆盘显示高清晰度地图纹理（使用已有 `proceduralEarthTex()` 或 Canvas 绘制，清晰度 ≥ 第一界面地球纹理）
- [ ] 穹顶保留半透明面片（供太阳轨迹投影），但**去除 wireframe 网格线**
- [ ] 去除 30°/60° 高度参考圈（或改为极淡、用户不感知的水平）
- [ ] 方位角标签（北/东/南/西等）保留且清晰可读

### AC-3: 太阳低于地平面时隐藏

- [ ] `sinAlt <= 0`（太阳高度角 ≤ 0°）时，太阳光点完全隐藏
- [ ] `sinAlt > 0` 时太阳光点正常显示在天球穹顶表面
- [ ] 极夜场景：整个太阳轨迹 + 太阳光点均隐藏（已实现，验证仍正常）

### AC-4: 鸭子动态阴影

- [ ] 鸭子下方地面有圆形阴影（类似第一界面 `updateDuckShadow()`）
- [ ] 阴影方向与当前太阳方位角一致（背离太阳）
- [ ] 太阳高度角越低阴影越长/越淡，太阳在天顶时阴影最小
- [ ] 太阳低于地平面时阴影隐藏

### AC-5: 主界面不受影响

- [ ] 第一界面所有功能正常（3D 地球、鸭子、轨迹、滑块、指南）
- [ ] 第二界面进入/退出正常
- [ ] 时刻滑块双向同步
