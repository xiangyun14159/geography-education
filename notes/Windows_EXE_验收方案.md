# Windows EXE 打包验收方案

> 目标：将「日地天文教学工具」打包为 Windows exe，兼容 Win10 / Win11。
> 原则：不动核心代码（index.html / three.min.js 只读），仅添加桌面壳层。

---

## 一、技术方案

| 项目 | 选型 |
|------|------|
| 桌面壳 | Electron（最稳定、社区最大、Win10/Win11 原生支持） |
| 打包工具 | electron-builder |
| 核心资产 | `index.html` + `three.min.js`（原样复制，不修改） |
| 窗口配置 | 可拖拽边框调整大小、默认 1280x800、支持全屏 |
| 图标 | 复用 Android ic_launcher 或生成默认图标 |

## 二、验收条件

- [x] `index.html` 与源文件逐字节一致（SHA256: `a36821e8...d4255` 两端匹配）
- [x] `three.min.js` 与源文件逐字节一致（SHA256: `ec0a8437...fab7e` 两端匹配）
- [x] 所有新增文件放在 `desktop/` 独立目录，`src/` 完全未触碰

### AC-2: 程序可启动（Win10 / Win11）⚠️ 需手动验证

- [ ] 双击 exe 可启动，出现独立窗口
- [ ] 窗口标题为「日地天文教学」
- [ ] 窗口默认尺寸合理（1280x800），可拖拽边框调整
- [ ] 无需管理员权限即可运行

### AC-3: 核心功能验证 ⚠️ 需手动验证

- [ ] 3D 地球渲染正常（NASA Blue Marble 纹理加载，非黑球）
- [ ] 鼠标拖拽旋转视角正常
- [ ] 鼠标滚轮缩放正常
- [ ] 底部控制面板可见，所有滑块可交互
- [ ] 四季预设按钮（春分/夏至/秋分/冬至）可点击切换
- [ ] 鸭子模型在地球表面正确显示
- [ ] 太阳视运动轨迹（黄色圆环）正常渲染
- [ ] 晨昏线正常显示
- [ ] 左上角天文信息面板正常更新
- [ ] 使用指南（? 按钮）可弹出和关闭

- [ ] 右上角信息栏显示正常

### AC-4: 离线运行

- [x] HTML 源码中无任何外部 HTTP/HTTPS 引用（已 grep 验证）
- [x] 所有资源（three.js、纹理）均为本地加载

### AC-5: 稳定性 ⚠️ 需手动验证

- [ ] 连续操作 5 分钟不崩溃
- [ ] 窗口最大化/还原切换正常
- [ ] 窗口关闭后进程正常退出（无残留进程）

### AC-6: 打包产物

- [x] 生成 `日地天文教学_v1.0_win64.exe`（71MB，PE32 executable，NSIS portable）
- [x] 便携版 exe，双击即用，无需安装运行时（Electron 内嵌 Chromium）
- [x] 产物放在 `outputs/` 目录

---

## 三、实施步骤

1. 在项目根创建 `desktop/` 目录，初始化 Electron 项目
2. 编写 `main.js`（Electron 主进程，~30行）
3. 编写 `package.json`（依赖 electron + electron-builder）
4. 复制 `index.html` + `three.min.js` 到 `desktop/app/`（原样，不修改）
5. 配置 electron-builder 打包参数
6. 本地构建验证
7. 打包生成 exe 安装包
8. 按本方案逐项验收

---

## 四、不做的

- 不修改 `index.html` 任何代码
- 不修改 `three.min.js`
- 不引入额外 JS 框架
- 不添加网络依赖
- 不修改项目已有的 src/ outputs/ docs/ 结构
