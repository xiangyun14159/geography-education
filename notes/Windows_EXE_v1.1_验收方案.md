# v1.1 第二界面修改验收方案

> 修改范围：仅 `index.html` 中 `buildCelestialScene()` / `updateCelestialSunDot()` 及相关函数
> 原则：不动第一界面代码、不动 three.min.js

---

## AC-1: 第二界面使用真实鸭子模型

- [ ] 中心位置显示与第一界面完全相同的黄色小鸭（身体、头、喙、眼、翅膀、尾）
- [ ] 鸭子朝向：头 = 北(+Z)、尾 = 南(-Z)、左 = 西(-X)、右 = 东(+X)
- [ ] 鸭子位于地面 y≈0 处，底面与地面齐平
- [ ] 鸭子在 Phong 材质光照下正确显示明暗

## AC-2: 界面简洁 + 底面高清晰地图

- [ ] 去除穹顶 wireframe 网格线
- [ ] 去除 30°/60° 高度参考圈
- [ ] 地面圆盘替换为高清晰地球卫星纹理（1024x512 canvas 绘制）
- [ ] 纹理以鸭子经纬度为中心投影（方位等距投影或简单偏移）
- [ ] 方位角标签（北/东北/东/东南/南/西南/西/西北）保留且清晰可读
- [ ] 地平线圈保留

## AC-3: 太阳在地平线以下时完全隐藏

- [ ] sinAlt < 0 时 `sunDot2.visible = false`（太阳光点隐藏）
- [ ] sunDot2 及其子元素（glow sprite）一起隐藏
- [ ] 太阳在地平线以上时正常显示（visible=true, opacity=1）

## AC-4: 实时鸭子影子

- [ ] 鸭子下方地面上显示圆形阴影
- [ ] 阴影方向与太阳位置相反
- [ ] 太阳高度角越高阴影越深/越小，越低阴影越浅/越大
- [ ] 太阳在地平线以下时阴影隐藏
- [ ] 随时刻滑块拖动实时更新

## AC-5: 原有功能不受影响

- [ ] 第一界面所有功能正常
- [ ] 第二界面返回按钮、时刻滑块正常
- [ ] 鼠标/触控旋转缩放正常
