# v3.26 验收标准

## AC-1: 地球纹理正常渲染（根治黑球）

**根因分析**：
v3.25 黑球的根因是 `<img>` 标签在 Android WebView 中不可靠：
1. `display:none` 在部分 WebView 版本中阻止图片加载，`naturalWidth` 始终为 0
2. 即使图片加载成功，`canvas.drawImage(file:// 来源)` 可能污染 canvas，WebGL 拒绝上传污染纹理 → 黑球
3. `earthMat.map` 被设置为损坏的 CanvasTexture 后，3 秒超时保护无法触发（`earthMat.map` 非空）

**修复方案**：
- 彻底移除 `<img>` 标签 + IIFE 异步加载方案
- 回归 v3.19/v3.24 验证过的同步 procedural 纹理方案
- 升级 procedural 纹理配色为 NASA 风格（更丰富的蓝色海洋 + 绿棕大陆）
- 删除无用 `earth_tex.jpg`
