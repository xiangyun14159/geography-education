# v3.22 验收标准

## AC-1: 鸭子南北半球均不变形 ✅

**根因**：`makeBasis(east, up, north)` 中 `east × up = -north`（已验证），构成左手坐标系。Three.js `setFromRotationMatrix` 要求右手坐标系矩阵，左手矩阵导致四元数包含镜像反射，鸭子模型被压扁。

**修复**：`index.html:647` — `makeBasis(east.clone().negate(), up, north)` → `(-east) × up = north` 恢复右手系。

**验收条件**：
- [x] `placeDuck()` 中 `makeBasis` 使用右手坐标系基向量（`east.negate()`）
- [x] 鸭子在南半球（纬度 -90°~0°）不再变扁
- [x] 鸭子始终垂直于地球表面，底部接触球面
- [x] 调节纬度滑块时鸭子形态保持稳定
