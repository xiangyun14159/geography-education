# v3.27 验收标准

## AC-1: 标准世界地图纹理（base64 嵌入，根治黑球）

**根因回顾**：
v3.25/v3.23 `<img src="file://">` 在 Android WebView 中不可靠：
- `display:none` 阻止加载 / canvas 污染 / MAX_TEXTURE_SIZE 超限
- 两次尝试均失败（PNG + JPEG）

**修复方案**：
- 下载 NASA Blue Marble 标准地图，缩放至 1024x512 JPEG（~105KB，quality 90）
- 转为 base64 data URL，直接嵌入 JS 代码（140KB 字符串）
- `new Image()` + data URL 加载——无 file:// 依赖、无跨域问题、无 canvas 污染
- 保留 procedural 降级（belt and suspenders）

**验收条件**：
- [x] 地球纹理为标准 NASA Blue Marble 卫星图（蓝海 + 绿棕大陆 + 白云）
- [x] 纹理通过 base64 data URL 加载，无 `<img>` DOM 标签
- [x] 纹理分辨率为 1024x512（清晰度较 v3.27 初版 512x256 提升 4 倍）
- [x] 加载失败时降级到 procedural 纹理
- [x] 无外部图片文件依赖（无 earth_tex.jpg/png）
